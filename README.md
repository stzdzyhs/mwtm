# mwtm
在多显示器的环境中，移动窗口到指定的显示器

编译 
请参见INSTALL

运行参数
mwtm 'win0Title' monitorName 'win1Title' monitorName ...

此工具参考了wmctrl, xrandr的代码，在此一起表示感谢！
